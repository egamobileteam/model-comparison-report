Methodology — Transparency Note

Known limitations of this comparison

(a) Claude Code usage across Team seats

Claude Code is included on both Team Standard and Team Premium seats — no add-on required. The difference between the two seat types is usage capacity, not feature access: Premium seats provide roughly 5× the usage of Standard seats. Organisations can mix seat types, assigning Premium to developers whose Claude Code workloads need the higher allowance.

(b) The tool vs the AI behind it

Scores partly reflect the AI model powering each tool, not just the tool itself. When the underlying AI is updated, results may change regardless of the product.

(c) PoC vs production scale

PoC task complexity may not surface differences that emerge at production scale, across larger codebases and longer engagements.

(d) Category mismatch

These tools are not the same kind of product — a coding-editor assistant, a browser-based coding workspace, a self-directed agent, and a full platform. Some criteria naturally favour one type over another.

(e) Shared usage limits

Activity inside Claude Design draws from your plan's shared pool of usage limits. Every prompt, design iteration, and preview counts toward the same session allowances you use for regular chats, Claude Code, and Cowork — there is no separate Claude Design quota. Heavy design work can exhaust capacity for coding and chat within the same billing period. Source: Anthropic Help Center, verified July 2026.

Recommendation for Adoption Status: Complete
Recommended Platform
Claude Platform
Business Justification
Claude Platform earned the highest overall score (8.5/10) across all eight categories. For a government digital programme that needs Arabic right-to-left support, consistent branding, and strong security and governance, Claude Platform offers the strongest foundation: a built-in coding assistant, compliance and audit tools, and organisation-wide policy controls. Because it can keep the entire brief in mind at once, it made fewer mistakes and needed less rework — even though it took slightly longer to finish.
Key Strengths
  • Best at tracking every requirement, and top-rated for enterprise readiness (10/10) with strong compliance and organisation-wide controls
  • Delivered a complete Arabic-ready homepage with an Arabic/English switch, proper Arabic fonts, and automated checks
  • Produced clean, safe code with nothing left over, and needed the fewest do-overs of all platforms tested
  • The coding assistant is included on every Team seat; mix Standard and Premium seats purely by how much usage each person needs, giving Premium (5× the usage) to heavy users
Risks & Mitigations
  • Usage capacity, not licensing — Claude Code is included on both Standard ($20–25/seat/mo) and Premium ($100–125/seat/mo) seats; budget Premium only where developers need the 5× usage allowance
  • Slower initial delivery (3h 28m vs Replit's 1h 52m) — Reduce this by running tasks in parallel and setting up reusable project templates in advance
  • Command-line interface limits access for non-developers — Pair it with Claude's regular chat interface for Product and Design teams
  • Dependent on the AI model — Scores reflect today's models; re-evaluate every quarter as they improve
  • May need a second tool — Cursor is recommended as a complementary code editor for developers who prefer a traditional editor (scored 8.2/10)
